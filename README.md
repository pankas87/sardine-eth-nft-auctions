# README

## System Requirements

* [rvm](https://rvm.io/): Ruby Version Manager (or Ruby v3.1.0 with Bundler) 
* [SQLite](https://www.sqlite.org/index.html) (Mandatory)

## Installation

Execute the `install.sh` bash script
```bash
sh ./install.sh
```

## Starting up the server

```bash
bundle exec rails s -p 8080
```

## Test Suite

```bash
bundle exec rspec
```
